PsiHub

Package hub client.

psihub is the local-first client for validating PSI package metadata, publishing packages to a local hub, downloading them, and rendering cards for humans and agents.

It does not launch services. It keeps packages inspectable and portable so psi, AAAX, scripts, or agents can decide how to run them.

Install Install the package hub client. 
python -m pip install psihub
Initialize And Validate Create package metadata and check package shape before sharing. 
psihub init demo-package --org demo --name echo --kind tactic
psihub validate demo-package
Publish Locally Publish copies a normal package folder into deterministic local hub storage. 
psihub --hub .psihub publish demo-package --local
psihub --hub .psihub list
Download Download returns another ordinary package folder with docs, examples, assets, and psi.toml. 
psihub --hub .psihub get demo/echo --dest ./downloaded
Explain Cards expose package resources and launch hints without printing secret values. 
psihub --hub .psihub card demo/echo
psihub --hub .psihub agent-card demo/echo
psihub --hub .psihub config-template demo/echo
Serve A Local API Expose the local hub to apps or development tools when needed. 
psihub --hub .psihub serve --host 127.0.0.1 --port 8787
